Task Descibtion

You are part of a software engineering team tasked with developing a new social networking platform, similar in concept to Twitter. This platform aims to enable users to share updates, follow other users, and interact through likes and comments.

The system will be built using a microservices architecture, with each microservice responsible for a specific set of functionalities. The microservices will communicate with each other to allow user interaction.

User Stories

  • As a user, I want to be able to create an account and log in.
  • As a user, I want to be able to post updates (tweets).
  • As a user, I want to be able to follow other users.
  • As a user, I want to be able to see updates from users I follow.
  • As a user, I want to be able to like and comment on updates.
  • As a user, I want to be able to search for other users, tweets and hashtags.
  • As a user, I want to be notified when someone likes or comments on my tweets.
